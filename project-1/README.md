
# Revenue Forecasting and Performance Analysis of Vodafone Qatar (2021–2023)

## Project Summary

The project transforms Vodafone Qatar’s 2021‑2023 telecom performance data into a unified, ML‑ready dataset using NexusMax, Python, and Microsoft Fabric, and visualizes key KPIs in Power BI. This enables finance and strategy teams to forecast revenue more accurately, monitor churn, and make data‑driven investment decisions.

---

## Business Problem

Vodafone Qatar’s finance and strategy groups lacked a consolidated view of revenue, subscriber growth, churn, and network performance, making quarterly budgeting and growth planning uncertain. Fluctuations in churn and segment‑level behavior created risk that could not be quantified in real time.

---

## Objective

- Create a clean, enriched data foundation that supports quarterly revenue forecasting.
- Build interactive Power BI dashboards to surface revenue, churn, ARPU and network availability trends.
- Identify high‑impact segments and regions where churn reduction can protect recurring revenue.

---

## Tools & Technologies

- Python (pandas)
- NexusMax
- Microsoft Fabric
- Power BI
- SQL/Data Warehousing
- Regression modeling

---

## Project Workflow

- Ingest raw telecom data from Kaggle and internal sources.
- Clean, standardize and enrich the data (date table, KPI calculations) using Python and NexusMax.
- Load the transformed data into Microsoft Fabric for centralized storage and ML readiness.
- Develop regression models to forecast quarterly revenue and churn.
- Design Power BI dashboards to visualize KPI trends, segment performance, and forecast scenarios.

---

## Key Insights

- Total revenue grew 14 % QoQ to $108 K, driven primarily by the Doha region’s 72 M subscribers.
- High‑value mobile prepaid churn rose 0.3 % QoQ to above 2.5 %, threatening revenue stability.
- Network availability is extremely high at 99.95 %, providing a solid foundation for retention initiatives.
- Churn in Mobile Postpaid and Fixed Broadband segments (2.41 %–2.74 %) exceeds the overall churn rate, indicating an avoidable annual loss of $8–12 K.
